2 . 添加两个数字

给定两个代表两个非负整数的非空链表。这些数字以相反的顺序存储，并且它们的每个节点都包含一个数字。将两个数字相加并将总和作为链表返回。

您可以假设这两个数字不包含任何前导零，除了数字 0 本身。

---

示例 1：
![img.png](img.png)

输入： l1 = [2,4,3], l2 = [5,6,4]
输出： [7,0,8]
解释： 342 + 465 = 807。
示例 2：

输入： l1 = [0]，l2 = [0]
输出： [0]
示例 3：

输入： l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
输出： [8,9,9,9,0,0,0,1]


---
约束：

每个链表的节点数在范围内[1, 100]。
0 <= Node.val <= 9
保证该列表表示一个没有前导零的数字。
